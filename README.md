#  Front-End Learning Repository

Este repositório reúne estudos e práticas em **Front-End** , com foco no aprendizado gradual das tecnologias web e no fortalecimento da lógica e estrutura de interfaces.

##  Objetivo

- Aprender e consolidar os fundamentos do desenvolvimento Front-End  
- Praticar a construção de interfaces web  
- Registrar a evolução técnica ao longo do aprendizado  

## Conteúdo

O repositório inclui exemplos e exercícios relacionados a:

- HTML  
- CSS  
- JavaScript  
- Estruturação de páginas web  
- Estilização e layouts  
- Interações básicas no navegador  

##  Tecnologias

- HTML5  
- CSS3  
- JavaScript
